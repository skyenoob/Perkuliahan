sub step untuk yang udah ada repo nya pada github dan file nya siap untuk di push
atur dan arahkan ke directory file yang ingin dituju :

add keseluruhan project                          : git add .
git commit -m "(diisi bebas/sesuai materi)"      : contoh git commit -m "tes"
set branch dengan nama main                      : git branch -M main
push ke github                                   : git push -u origin main

ket :  jika ingin ke repo lain maka gunakan dulu : git remote add origin git@github.com:userNameGithubKau/repoYgDituju.git

baru gunakan command untuk push
